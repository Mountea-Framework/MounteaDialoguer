# Steam Auto-Cloud Setup (Mountea Dialoguer)

## Goal
Sync Steam provider snapshot files generated by the app:

- Relative data folder: `steam-sync/<profile-id>/`
- File type: `*.json`
- One file per synced project snapshot

## Actual runtime location
Electron writes under `app.getPath('userData')`:

- Windows: `%APPDATA%/Mountea Dialoguer/steam-sync/<profile-id>/`
- Linux: `~/.config/Mountea Dialoguer/steam-sync/<profile-id>/`
- macOS: `~/Library/Application Support/Mountea Dialoguer/steam-sync/<profile-id>/`

## Steamworks configuration
1. Open App Admin -> `SteamPipe` -> `Steam Cloud`.
2. Enable Auto-Cloud.
3. Add Root Paths:
   - Windows root override: `%APPDATA%/Mountea Dialoguer/steam-sync`
   - Linux root override: `~/.config/Mountea Dialoguer/steam-sync`
   - macOS root override: `~/Library/Application Support/Mountea Dialoguer/steam-sync`
4. Subdirectory pattern:
   - `*`
5. File pattern:
   - `*.json`
6. Recommended quota:
   - Byte quota per user: `1073741824` (1 GB)
   - File count per user: `5000`

## Notes
- Steam profile separation is handled by `<profile-id>` folder naming (`steam-<steamid64>`).
- Google sync remains optional and independent from Steam Auto-Cloud.
